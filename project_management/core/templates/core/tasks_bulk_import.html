Work-Type Specific Templates

Each work type has its own set of columns tailored to its needs:

Task

title, description, project, subproject, priority, due_date, assigned_to, file_attachment

Bug

title, project, severity, priority, status, steps_to_reproduce, expected_behavior, actual_behavior, file_attachment

Story

title, description, project, priority, story_points, acceptance_criteria, due_date, file_attachment

Defect

title, project, severity, priority, status, steps_to_reproduce, expected_behavior, actual_behavior, file_attachment

Subtask

title, description, project, parent_task_id, priority, due_date, assigned_to, file_attachment

Report

title, description, project, report_type, priority, due_date, assigned_to, file_attachment

Change Request

title, description, project, change_type, impact, priority, due_date, assigned_to, file_attachment

Important Tips:

  • Download the template for your specific work type to get the right columns
  • The work type is determined by which template you download (no work_type column needed)
  • All templates include dropdowns for projects, subprojects, and team members
  • file_attachment: Enter the full file path (e.g., C:\path\to\file.png) to attach files
  • Files will be automatically uploaded and attached to the task during import
  • Dates must be in YYYY-MM-DD format (e.g., 2026-03-15)
  • Supported formats: Excel (.xlsx) with data validation
